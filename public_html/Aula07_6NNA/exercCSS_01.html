Comando CSS Efeito
p { color: red; }

O conteúdo de todos os parágrafos HTML (tag p) serão de cor vermelha
.textoInterno { font-size: 12pt; color: blue; } O conteúdo do elemento da classe “textoInterno” terá tamanho 12 e cor azul.
#textoExterno { font-family: Arial, font-weight: bold; } Formatação da fonte: tipo, tamanho e estilo
#divTopo { width: 90%;} O elemento com ID “divTopo” terá comprimento de 90% da tela.
h2 { padding-top: 40px; } O elementos do h2 terão espaçamentos de 40px entre o conteúdo e as bordas
